```python

N, K = map(int, input().split())
coin=[]
for _ in range(N):
    tmp = int(input())
    coin.append(tmp)

cnt =0
while 1 :
    for i in range(N-1,-1,-1):
        if K >= coin[i] :
            cnt += K//coin[i]
            K = K%coin[i]
    if K == 0 :
        break
print(cnt)
```
